Projects

Smart Energy: Predictive Modelling for Optimized Consumption

Developed a predictive model for energy consumption in smart cities using time series analysis (XGBoost and ARIMA) and machine learning regression techniques. Incorporated factors like weather, holidays, and time patterns for accurate forecasts. Used LIME and SHAP for explainability, enabling better decision-making for real-time energy monitoring and adaptive management.

Lake Encroachment Detection

Implemented K-means clustering on satellite data from Google Earth Engine to analyze land use changes. Set up an automated email alert system to notify authorities upon detecting unauthorized land use.

Cyber-bullying Detection in Social Media Using Machine Learning

Developed a model to detect cyberbullying in online text using machine learning and NLP. Analyzed hate speech-related tweets and Wikipedia comments, evaluating multiple feature extraction techniques and classifiers. Achieved over 90% accuracy on Twitter data and above 80% on Wikipedia data.
